# 實驗 5：使用 Copilot（預覽版）創建 AI 生成的網站、網頁並將其他組件添加到頁面

**注意：**使用 Copilot 創建 Power Pages
網站時，請務必注意，每次創建網站時，Copilot
可能會生成不同的模板或佈局。結果：

\- 生成的網站的確切佈局和設計可能因實例而異。

\-
在按照指南或教程中的創建步驟進行作時，請將它們用作參考，而不是期望獲得相同的結果。

\- 由於 Copilot
的動態生成功能，提供的屏幕截圖可能與您在實例中看到的屏幕截圖不同。

**提示：**
專注於理解整個流程和概念，而不是複製確切的設計，因為核心功能和步驟將保持一致。

## 練習 1：在 AI 的幫助下創建站點

要使用 Copilot 創建站點：

1.  使用 [**https://make.powerpages.microsoft.com/**](urn:gd:lg:a:send-vm-keys)
    轉到 Power Pages

2.  確保您位於 Developer environment - **Dev One** 中，然後單擊 **Get
    started**。

![](./media/image1.png)

3.  選擇 **Skip** on **Tell us about yourself** **窗口**。

![A screenshot of a computer Description automatically
generated](./media/image2.png)

4.  現在，您將導航到 Power Pages
    門戶的主頁。輸入要構建的站點的描述，或選擇該站點的建議描述。然後，按
    輸入 鍵或選擇文本框右下角的紙飛機圖標。

示例描述：

- [**Build a website for public transportation for residents of city, to
  view routes and fares.**](urn:gd:lg:a:send-vm-keys)

![](./media/image3.png)

5.  Copilot 會根據您的描述生成站點名稱和 Web 地址。保持建議的名稱和 Web
    地址不變，然後選擇 **Next**。

> ![](./media/image4.png)

6.  在 “Choose a layout page’” 中，選擇**Next**。然後，在 “添加常用頁面”
    頁面上，選擇所有網頁，然後單擊 “完成” 。在 Introducing Copilot in
    power 頁面上，單擊 **Next**，最後選擇 **Done**。

![A screenshot of a computer Description automatically
generated](./media/image5.png)

![A screenshot of a computer Description automatically
generated](./media/image6.png)

![](./media/image7.png)

7.  單擊“啟用副駕駛”，然後在“將副駕駛添加到站點”頁面的下一頁上，打開“在坐下時啟用副駕駛”

### 練習 2：使用 Copilot 創建 AI 生成的網頁（**預覽版**）

### 任務 1：生成網頁

要使用 Copilot 生成網頁：

1.  從左側導航窗格中，選擇 **Pages**
    工作區，然後您將進入新創建的網站的主頁

2.  最初，您將位於 **Pages** 工作區。

![](./media/image11.png)

3.  如果 Copilot 面板尚未打開，請選擇命令欄中的 Copilot 按鈕。

• 在 Choose a starter prompt， request an action， or ask a question
文本字段中鍵入網頁的描述。

示例描述：

- [**Create a Thank you for visiting our site
webpage**](urn:gd:lg:a:send-vm-keys)

![](./media/image12.png)

要完成頁面審核，請選擇 **Keep it** （保留）。

![](./media/image13.png)

生成頁面後，它會添加到主導航中，您可以查看頁面和內容。頁面中的部分、文本和圖像可使用
WYSIWYG 編輯器進行低代碼編輯。

![](./media/image14.png)

## 練習 3：使用 Copilot 添加 AI 生成的表單（預覽）

### 任務 1：使用 Copilot 創建表單

1.  轉到 **Pages 工作區**，然後選擇表單的 **Routes** 頁面。

![](./media/image15.png)

2.  將鼠標懸停在 Routes 網頁的第一個部分上，然後單擊 **+ Add a section
    \> 1Column**。

![](./media/image16.png)

3.  選擇 **More** ，然後選擇 **Form** 組件。

![](./media/image17.png)

4.  在 Describe a form to create it （描述用於創建表單）
    下的文本框中，描述您的表單。您可以在描述中使用最多 250
    個字符。要將描述發送給 Copilot，請按 Enter
    鍵或選擇文本框右下角的紙質飛機圖標。

**例：** [**Create a form for collecting suggestions and
addition/deletion of the routes**](urn:gd:lg:a:send-vm-keys)

![](./media/image18.png)

**注：** 您可以檢查描述右側的預覽，並根據需要優化表單。

- 要更改表單，請選擇快速作或優化描述。

- 歷史記錄顯示您到目前為止輸入的描述。

- 選擇 “**Start over**” 以擦除所有內容並從新描述開始。

![](./media/image19.png)

5.  要將表單添加到頁面，請選擇 **OK**。

![](./media/image20.png)

### 任務 3：使用 Copilot 生成文本

1.  轉到 **Pages 工作區**，然後選擇表單的 **About Us** page
    （關於我們頁面）。

![](./media/image26.png)

2.  將鼠標懸停在頁面上，然後在我們的服務部分下選擇 **+Add a section \>
    1Column** 。

![](./media/image27.png)

3.  選擇 **Text** 組件。

![](./media/image28.png)

4.  在文本組件工具欄中，選擇 **Copilot** 圖標。

![](./media/image29.png)

5.  描述您要讓 Copilot 生成的文本; 例如， " [**I want to describe
    importance of local transport**](urn:gd:lg:a:send-vm-keys)".
    您可以在描述中使用最多 1500 個字符。要將描述發送給 Copilot，請按
    Enter 鍵或選擇文本框右下角的紙質飛機圖標。

![A screenshot of a computer Description automatically
generated](./media/image30.png)

6.  檢查描述下方的 **preview** 並根據需要優化文本。

    - 要優化描述並生成新文本，請選擇 **Rewrite** （重寫）。

    - 要保留生成的文本，但使其聽起來更友好、更隨意、更有教育意義、更專業或更興奮，請選擇
      **Change the tone**（更改語氣）。

    - 要保留生成的文本，但使其更簡潔或對其進行擴展，請選擇 **Adjust the
      length**（調整長度）。

    - 要向 Copilot 提供更多信息，請選擇 Add more details for Copilot to
      work with。

    - 要向 Microsoft 提供有關 Copilot
      如何解釋您的描述或優化的反饋，請選擇豎起大拇指或豎起大拇指圖標。

7.  要將文本添加到頁面，請選擇 **Add to page**。

![](./media/image31.png)

### 任務 4：測試您的網站

1.  選擇 **Preview**，然後選擇
    **Desktop**。如果系統提示，請使用管理員憑據登錄，然後選中 **checkbox
    of the consent**，然後選擇 **Accept** （接受）。

![A screenshot of a web page Description automatically
generated](./media/image32.png)

2.  您可以查看主頁和其他網頁的列表。

![](./media/image33.png)

3.  選擇 Router page。您可以觀察 AI 生成的表單。

![A screenshot of a route Description automatically
generated](./media/image34.png)

4.  填寫表單的幾個字段，然後選擇表單末尾的 **Submit** 按鈕。

![A screenshot of a route Description automatically
generated](./media/image35.png)

5.  選中 **checkbox of the consent** ，然後選擇 **Accept**。

![](./media/image36.png)

6.  成功提交表單後，請返回設計工作室。

**注意：**如果您沒有收到提交成功的消息，請再試一次。

![A screenshot of a computer Description automatically
generated](./media/image37.png)

7.  選擇 **Data**，然後選擇 **Copilot Suggestion Routes**
    表（表名稱可能不同），並確認您可以看到記錄是使用多步驟表單流程創建的。

![A screenshot of a computer Description automatically
generated](./media/image38.png)
