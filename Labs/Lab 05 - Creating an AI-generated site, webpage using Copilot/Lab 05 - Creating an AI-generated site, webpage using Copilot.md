# Atelier 5 : Création d'un site ou d'une page Web générés par l'IA à l'aide de Copilot (aperçu) et ajout d'autres composants aux pages

**Remarque :** Lors de la création d'un site Web Power Pages à l'aide de
Copilot, il est important de noter que Copilot peut générer un modèle ou
une mise en page différent à chaque fois que le site Web est créé. En
conséquence :

\- La mise en page et la conception exactes du site Web généré peuvent
varier d'un cas à l'autre.

\- Tout en suivant les étapes de création dans les guides ou les
tutoriels, utilisez-les comme référence plutôt que d'attendre des
résultats identiques.

\- Les captures d'écran fournies peuvent différer de ce que vous voyez
dans votre instance en raison des capacités de génération dynamique de
Copilot.

**Conseil** : concentrez-vous sur la compréhension du processus et des
concepts globaux plutôt que sur la reproduction exacte de la conception,
car les fonctionnalités et les étapes de base resteront cohérentes.

## Exercice 1 : Créer un site à l'aide de l'IA

Pour utiliser Copilot pour la création de sites :

1.  Accédez à Power Pages à l'aide de
    [**https://make.powerpages.microsoft.com/**](urn:gd:lg:a:send-vm-keys)

2.  Assurez-vous que vous êtes dans l'environnement du développeur -
    **Dev One**, puis cliquez sur **Get Started**.

![](./media/image1.png)

3.  Sélectionnez **Skip** à la **Tell us about yourself window**.![Une
    capture d'écran d'un ordinateur Description générée
    automatiquement](./media/image2.png)

4.  Maintenant, vous serez redirigé vers la page d'accueil du portail
    Power Pages. Entrez une description du site que vous souhaitez créer
    ou sélectionnez une description suggérée du site. Ensuite, appuyez
    sur la touche Entrée ou sélectionnez l'icône de l'avion en papier
    dans le coin inférieur droit de la zone de texte.

Exemple de description :

- [**Build a website for public transportation for residents of city, to
  view routes and fares.**](urn:gd:lg:a:send-vm-keys)

![](./media/image3.png)

6.  Dans « Choisir une page de mise en page », sélectionnez **Next**.
    Ensuite, sur la page « Ajouter des pages communes », sélectionnez
    toutes les pages Web puis cliquez sur Terminé. Dans les pages
    Présentation de Copilot dans Power, cliquez sur **Next** et enfin
    sélectionnez **Done**.

### Exercice 2 : Création d'une page Web générée par l'IA à l'aide de Copilot (**aperçu**)

### Tâche 1 : Générer une page web

Pour utiliser Copilot afin de générer une page Web :

1.  Dans le volet de navigation de gauche, sélectionnez Espace de
    travail **Pages**, puis vous serez sur la page d'accueil de votre
    site Web nouvellement créé

2.  Au départ, vous serez sur l'espace de travail **Pages**.

![](./media/image11.png)

3.  Sélectionnez le bouton Copilot dans la barre de commandes, si le
    panneau Copilot n'est pas déjà ouvert.

• Saisissez une description de votre page Web dans le champ de texte
Choisir une invite de démarrage, Demander une action ou Poser une
question.

Exemples de descriptions :

 [**Create a Thank you for visiting our site
webpage**](urn:gd:lg:a:send-vm-keys)

![](./media/image12.png)

Pour terminer l'examen de la page, sélectionnez **Keep it**.

![](./media/image13.png)

Une fois la page générée, elle est ajoutée à la navigation principale et
vous pouvez examiner la page et le contenu. Les sections, le texte et
les images de la page sont modifiables à l'aide de l'éditeur WYSIWYG.

![](./media/image14.png)

## Exercice 3 : Ajouter un formulaire AI-generated à l'aide de Copilot (préversion)

### Tâche 1 : Créer un formulaire avec Copilot

1.  Accédez à le **Pages workspace**  et sélectionnez une page
    **Routes** pour votre formulaire.

![](./media/image15.png)

2.  Passez la souris sur la première section de la page Web Itinéraires,
    puis cliquez sur **+ Add a section** \> **1Column.**

![](./media/image16.png)

3.  Sélectionnez **More**, puis Composant de **form**.

![](./media/image17.png)

4.  Dans la zone de texte sous Décrire un formulaire pour le créer,
    décrivez votre formulaire. Vous pouvez utiliser jusqu'à 250
    caractères dans votre description. Pour envoyer votre description à
    Copilot, appuyez sur la touche Entrée ou sélectionnez l'icône d'un
    avion en papier dans le coin inférieur droit de la zone de texte.

**Exemple** :**Create a form for collecting suggestions and
addition/deletion of the routes**

![](./media/image18.png)

**Remarque :** Vous pouvez vérifier l'aperçu à droite de votre
description et affiner le formulaire si nécessaire.

- Pour modifier le formulaire, sélectionnez une action rapide ou affinez
  votre description.

- L'historique vous montre les descriptions que vous avez saisies
  jusqu'à présent.

- Sélectionnez **Start over**  pour tout effacer et commencer avec une
  nouvelle description.

![](./media/image19.png)

5.  Pour ajouter le formulaire à la page, sélectionnez **OK.**

![](./media/image20.png)

### Tâche 3 : Générer du texte avec Copilot

1.  Accédez à la **Pages workspace**  et sélectionnez la **About Us** de
    nous pour votre formulaire.

![](./media/image26.png)

2.  Passez la souris sur la page et sélectionnez **+ Add a
    section** \> **1Column**  sous la section Nos services.

![](./media/image27.png)

3.  Sélectionnez le **composant Texte**.

![](./media/image28.png)

4.  Dans la barre d'outils du composant de texte, sélectionnez l' icône
    **Copilote**.

![](./media/image29.png)

5.  Décrivez le texte que vous souhaitez que Copilot génère, par
    exemple, " [**I want to describe importance of local
    transport**](urn:gd:lg:a:send-vm-keys)". Vous pouvez utiliser
    jusqu'à 1 500 caractères dans votre description. Pour envoyer votre
    description à Copilot, appuyez sur la touche Entrée ou sélectionnez
    l'icône d'un avion en papier dans le coin inférieur droit de la zone
    de texte.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image30.png)

6.  Vérifiez le **preview** sous votre description et affinez le texte
    si nécessaire.

    - Pour affiner votre description et générer un nouveau texte,
      sélectionnez **Rewrite**.

    - Pour conserver le texte généré mais le rendre plus convivial,
      décontracté, éducatif, professionnel ou enthousiaste, sélectionnez
      **Change the tone**.

    - Pour conserver le texte généré mais le rendre plus concis ou le
      développer, sélectionnez **Adjust the length**.

    - Pour fournir plus d'informations à Copilot, sélectionnez
      select **Add more details for Copilot to work with**.

    - Pour donner à Microsoft des commentaires sur la façon dont Copilot
      a interprété votre description ou votre amélioration, sélectionnez
      l'icône du pouce vers le haut ou vers le bas.

7.  Pour ajouter le texte à la page, sélectionnez **Add to page**.

![](./media/image31.png)

1.  Sélectionnez **Preview**,, puis **Desktop**.. Si vous y êtes invité,
    connectez-vous avec les informations d'identification
    d'administrateur, cochez la **checkbox of the consent** , puis
    sélectionnez **Accept**.

![Une capture d'écran d'une page web Description générée

2.  Vous pouvez voir la page d'accueil et la liste des autres pages Web.

![](./media/image33.png)

3.  Sélectionnez la page Routeur. Vous pouvez observer le formulaire
    généré par l'IA.

![Une capture d'écran d'un itinéraire Description générée
automatiquement](./media/image34.png)

4.  Remplissez les quelques champs du formulaire, puis sélectionnez le
    bouton **Submit** à la fin du formulaire.

![Une capture d'écran d'un itinéraire Description générée
automatiquement](./media/image35.png)

5.  Cochez la **checkbox of the consent**  et sélectionnez **Accept**.

![](./media/image36.png)

6.  Une fois que vous avez soumis le formulaire avec succès, retournez
    au studio de design.

**Remarque** : Essayez une fois de plus si vous ne recevez pas de

    du processus de formulaire en plusieurs étapes.
