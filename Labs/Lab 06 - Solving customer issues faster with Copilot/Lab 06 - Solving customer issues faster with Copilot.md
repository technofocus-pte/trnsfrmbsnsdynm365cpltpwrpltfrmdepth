## Exercice 1 : Résumer les cas

Les résumés de cas Copilot vous aident à comprendre rapidement le
contexte d'un cas et à résoudre plus efficacement les problèmes des
clients. Le résumé du cas comprend des informations clés telles que le
titre du cas, le client, l'objet, le produit, la priorité, le type de
cas et la description.

## Exercice 2 : Brouillon de questions

1.  Dans le **Customer Service Agent Dashboard**  sélectionnez l'un des
    exemples de cas, par exemple, **A Mineral Build Up in Water Supply**

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image9.png)

Vous pouvez poser des questions libres tout comme vous le feriez pour
votre collègue ou votre superviseur qui pourrait connaître les réponses.

2.  Dans le volet **Copilot**, qui s'ouvre sur le côté droit de l'écran,
    tapez [**Qu'est-ce que le minéral accumulé dans l'approvisionnement
    en eau ?**](urn:gd:lg:a:send-vm-keys) et sélectionnez **l**'icône
    **Send**.

> **Remarque** : Le processus de configuration de Copilot peut prendre
> un certain temps. Pour optimiser votre session de laboratoire,
> enregistrez votre progression et passez à l'activité de laboratoire
> suivante. Vous pouvez revenir à cet atelier une fois la configuration
> de Copilot terminée pour reprendre là où vous vous étiez arrêté.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image10.png)

3.  Vous pouvez maintenant voir que **Copilot** a répondu à votre
    question.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image11.png)

4.  Vous pouvez poser plus de questions comme, que faire........ ou
    comment résoudre le problème de colmatage dans le système d'infusion
    intelligent ou  [**What to do if water doesn't come consistently in
    smart brew system?**](urn:gd:lg:a:send-vm-keys) et cliquez sur
    l'icône **Send. Copilot** répondra à votre question.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image12.png)

5.  Avec **Copilot**, vous pouvez effectuer les actions suivantes :

    - **Posez une question directe** : Copilot affiche la réponse la
      plus pertinente à partir des sources de connaissances que votre
      organisation a mises à disposition.

    - **Posez des questions de suivi au tour par tour** : Si la réponse
      de Copilot n'est pas immédiatement utile, vous pouvez poser des
      questions de suivi et guider Copilot de manière naturelle et
      conversationnelle.

    - **Demandez à Copilot d'essayer une meilleure réponse** : Copilot
      peut également reformuler les réponses en fonction de plus de
      conseils.

Par exemple**, [Can you summarize your
response?](urn:gd:lg:a:send-vm-keys)** ou [**Can you attempt a response
providing details for each of the steps you
mentioned?**](urn:gd:lg:a:send-vm-keys)

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image13.png)

6.  Si vous êtes satisfait de la réponse fournie par Copilot, vous
    pouvez utiliser la réponse entière ou une partie de celle-ci pour
    répondre à la question du client :

    - Copiez une partie de la réponse de Copilot dans votre chat ou
      lisez-la lors d'une conversation vocale. Sélectionnez l' icône
      **Copier** pour copier l'intégralité de la réponse dans le
      presse-papiers.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image14.png)

**Remarque** : Lorsque vous êtes dans une conversation de messagerie
numérique active, sélectionnez **Send to customer** pour ouvrir une
fenêtre de modification dans laquelle vous pouvez modifier la réponse et
l'envoyer au client. Vous pouvez également modifier les mots-clés du
client pour inciter Copilot à générer une réponse plus précise.

7.  Sélectionnez **Check sources** to pour voir la base de connaissances
    ou les liens vers le site Web à partir duquel Copilot a tiré la
    réponse. Vous pouvez utiliser ces informations supplémentaires comme
    ressource ou les partager avec le client.

![Une capture d'écran d'un écran d'ordinateur Description générée
automatiquement](./media/image15.png)

8.  Cliquez sur le lien et vous pouvez voir le contenu sur le côté
    gauche.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image16.png)

9.  Fermez l'onglet qui vient d'être ouvert. Ici, dans ce cas,
    'Dépannage des problèmes de brassage'.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image17.png)

10. Pour évaluer l'utilité des réponses de Copilot, sélectionnez l'icône
    pouce levé ou pouce baissé.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image18.png)

11. Une fois la conversation avec un client terminée, ou si vous
    souhaitez que Copilot ne tienne pas compte de la conversation
    jusqu'à présent, sélectionnez **Clear chat**  en haut du volet
    **Copilot**.

![Une capture d'écran d'un ordinateur Description générée

## Exercice 3 : Brouillon d'emails

1.  Sélectionnez **le Write an email**  dans le volet **Copilot**.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image20.png)

2.  Sur la page Vue d'ensemble du cas, sélectionnez Onglet **Related** ,
    puis sélectionnez **Activities**

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image21.png)

1.  Sélectionnez **+ New Activity \> Email**.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image22.png)

2.  Lorsque vous commencez à rédiger un e-mail, Copilot s'ouvre dans le
    panneau latéral droit et présente cinq invites prédéfinies et un
    prompt personnalisée :

    - **Suggérer un appel** : rédige une réponse qui suggère un appel
      avec le client aujourd'hui ou demain.

    - **Demander plus d'informations** : rédige une réponse qui demande
      plus de détails au client pour l'aider à résoudre le problème.

    - **Faire preuve d'empathie à l'égard des commentaires** : Rédige
      une réponse qui fournit une réponse empathique à un client qui
      exprime une plainte.

    - **Fournir des détails sur le produit/service** : Rédige une
      réponse qui offre des détails ou répond aux questions des clients
      sur un produit ou un service particulier.

    - **Résoudre le problème du client** : rédige une réponse qui
      fournit une résolution et des étapes ------ résolution, le cas
      échéant, au problème du client.

    - **Personnalisé** : vous permet de fournir votre propre prompt pour
      la réponse.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image23.png)

3.  Sélectionnez **Résoudre le problème du client** dans la liste des
    prompts prédéfinies.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image24.png)

4.  Vous pouvez voir, Copilot a généré une suggestion.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image25.png)

5.  Vous pouvez maintenant consulter la réponse. Apportez les
    modifications nécessaires, puis sélectionnez **Copy to email**  pour
    copier l'intégralité de la réponse à votre brouillon. Vous pouvez
    également sélectionner une partie de la réponse et utiliser le menu
    contextuel pour copier et coller la sélection.

![Une capture d'écran d'un téléphone Description générée
automatiquement](./media/image26.png)

6.  Réponse est maintenant disponible dans la partie du corps sur le
    côté gauche.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image27.png)

7.  Vous pouvez maintenant envoyer l'e-mail ou l'enregistrer.

![Une capture d'écran d'un ordinateur Description générée
automatiquement](./media/image28.png)
