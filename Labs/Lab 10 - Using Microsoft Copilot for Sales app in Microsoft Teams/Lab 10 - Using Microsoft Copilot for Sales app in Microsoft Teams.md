## Übung 3: Analysieren der Unterhaltungsdaten Ihrer Verkäufer mit Power BI (Vorschau)

1.  Um das Dashboard "Copilot for Sales – Conversation Intelligence"
    herunterzuladen, gehen Sie zu
    [**https://go.microsoft.com/fwlink/p/?linkid=2259835**](https://go.microsoft.com/fwlink/p/?linkid=2259835) Wählen
    Si **Get it now** aus.

![A screenshot of a computer Description automatically
generated](./media/image51.png)

2.  Sie werden zu Power BI-Apps navigiert. Wählen Sie im Fenster **"**
    Install this Power BI app' **"** die Option **Install** aus.

![A screenshot of a computer Description automatically
generated](./media/image52.png)

3.  Sie können das Copilot for Sales - Dashboard unter der Liste Apps
    sehen. Klicken Sie auf das **Copilot for Sales - Dashboard**, um es
    zu öffnen.

![A screenshot of a search engine Description automatically
generated](./media/image53.png)

4.  Die folgenden Seiten stehen in Ihrem Bericht zur Verfügung:

- Coaching opportunities

- Customer insights

- Call recordings

![A screenshot of a computer Description automatically
generated](./media/image54.png)

5.  Sie können den **Time period filter** und **Data filters** in Ihrem
    Bericht verwenden.

![](./media/image55.png)

6.  Wählen Sie im linken Navigationsbereich die Seite **Coaching
    opportunities** aus. Es zeigt Konversations-KPIs und
    Übersichtsdiagramme für den ausgewählten Zeitraum an.

![](./media/image56.png)

7.  Wählen Sie im linken Navigationsbereich die Seite **Customer
    insights** aus. Es zeigt die Kundenstimmung und die Top-Marken an,

![](./media/image57.png)

    recordings** aus. Diese Seite hilft Ihnen, sich Anrufaufzeichnungen
    anzuhören und den Kontext des Gesprächs zu verstehen.

![A screenshot of a computer Description automatically
generated](./media/image58.png)

### Aufgabe 2: Verbinden des Dashboards mit den Daten Ihrer Organisation

Wenn Sie das Dashboard "Copilot for Sales – Conversation Intelligence"
von Microsoft AppSource herunterladen, enthält es Beispieldaten. Um das
Dashboard mit Ihren eigenen Daten in Dataverse zu verbinden, gehen Sie
folgendermaßen vor:

1.  Wenn Sie davon ausgehen, dass das heruntergeladene Copilot for
    Sales-Dashboard bereits geöffnet ist, wählen Sie **Connect your
    data**.

![A screenshot of a computer Description automatically
generated](./media/image59.png)

2.  Geben Sie die folgenden Informationen ein:

a. **EnvironmentPath**: URL zu Ihrer Dataverse-Umgebung für die
Sales-Testversion. Sie müssen das Präfix https:// aus der eingegebenen
Umgebungspfad-URL entfernen. (Um die URL abzurufen, gehen Sie zum Power
Platform Admin Center. Wählen Sie im linken Navigationsbereich
Umgebungen aus, klicken Sie auf Verkaufstestversion und kopieren Sie die
Umgebungs-URL)

b. **CRM type**: Art des CRM, das Sie verwenden -- **Dynamics**.

c\. Wählen Sie **Next** aus.

![A screenshot of a computer Description automatically
generated](./media/image60.png)

3.  Wählen Sie eine Authentifizierungsmethode aus - **OAuth2** und
    Einstellungen für die Datenschutzstufe - **None** für Ihre
    Datenquelle. Wählen Sie **Sign in and connect** aus.

![A screenshot of a login page Description automatically
generated](./media/image61.png)

4.  Nachdem das Dashboard erfolgreich mit den Daten der Organisation
    verbunden wurde, können Sie den Bericht bearbeiten und
    veröffentlichen.
